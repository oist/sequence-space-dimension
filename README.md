This repository contains the data and code used in the paper "Descent from a common ancestor restricts exploration of protein sequence space"

Repository structure:

sequence-space-dimension/
├── README.md
├── LICENSE
├── environment.yml          
├── data/                     # summary tables, small example data
├── notebooks/                # Jupyter notebooks to reproduce all figures summary tables
├── params/                   # run parameters for simulations
├── src/                      # all source code
└── workflows/                # workflows the way they were run

Raw data can be found at Zenodo DOI:10.5281/zenodo.17623392
(description of the included files is in data/README.md)
